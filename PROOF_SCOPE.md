# Proof Scope

This proof kit is deliberately narrow. It shows the public-safe evidence story
for the `v1.6` warehouse robot safety prototype without publishing the private
engine.

## Included

- public overview of the consequence-memory warehouse safety claim
- locked `v1.6` headline metrics
- sanitized benchmark evidence card
- machine-readable public metrics summary
- verification notes and source hash references
- operator dashboard concept screenshot
- access, NDA, commercial-use, commercial-license, and licence boundary
  documents

## Excluded

- private source code
- notebook source
- simulation internals
- raw generated run folders
- dashboard implementation code
- private validation scripts
- private tests
- private deployment tooling
- customer data
- commercial production licence terms
- patent, trademark, database, hosted-service, or model-training rights

## Private Engine Boundary

The private engine is expected to remain in a separate private repository:

`consequence-warehouse-safety`

This public proof kit is expected to remain in a separate public repository:

`consequence-warehouse-safety-proof-kit`

The public proof kit may describe results produced by the private engine, but it
must not include enough implementation detail to reproduce or commercialize the
engine without permission.
